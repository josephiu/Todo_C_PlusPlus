# Todo_C_PlusPlus

This repo contains a todo list written in c++_ for research and learning purposes

# Overview
Todo_C_PlusPlus is a simple To-do-list application which allow user to creat a list of what they will like to do and can add more or remove any of the list they have created.

The aim of this application is to help me understand and get familar with c++ concepts such as: Variables, Conditions, Classes, Loops, Expressions, File read/write, functions etc.
